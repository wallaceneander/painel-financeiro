# 📱 GUIA DE INSTALAÇÃO - PAINEL FINANCEIRO APP

## ✅ OPÇÃO 1: PWA (Progressive Web App) - MAIS FÁCIL

### **PASSO 1: Hospedar os arquivos**

Você precisa colocar estes 4 arquivos em um servidor web:
1. `painel_financeiro_2026_v3.html`
2. `manifest.json`
3. `sw.js`
4. `icon-192.png` e `icon-512.png` (ícones do app)

**Opções de hospedagem GRATUITAS:**

#### A) GitHub Pages (Recomendado - Grátis)
1. Crie uma conta no GitHub (github.com)
2. Crie um repositório chamado "painel-financeiro"
3. Faça upload dos arquivos
4. Vá em Settings → Pages
5. Ative o GitHub Pages
6. Seu link será: `https://seu-usuario.github.io/painel-financeiro/painel_financeiro_2026_v3.html`

#### B) Netlify (Mais fácil - Grátis)
1. Acesse netlify.com
2. Arraste os arquivos na área de drop
3. Pronto! Link gerado automaticamente

#### C) Vercel (Grátis)
1. Acesse vercel.com
2. Faça upload dos arquivos
3. Link gerado automaticamente

---

### **PASSO 2: Instalar no iPhone**

1. Abra o Safari no iPhone
2. Acesse o link do seu painel
3. Clique no botão de **compartilhar** (🔗)
4. Role para baixo e clique em **"Adicionar à Tela de Início"**
5. Pronto! O app aparece na tela inicial 🎉

### **PASSO 3: Instalar no Android**

1. Abra o Chrome no Android
2. Acesse o link do seu painel
3. Clique nos 3 pontinhos (⋮)
4. Clique em **"Adicionar à tela inicial"**
5. Ou aparecerá um banner automático perguntando "Instalar app?"
6. Pronto! O app aparece na tela inicial 🎉

---

## 🎨 CRIAR OS ÍCONES

Você precisa criar 2 ícones para o app:
- `icon-192.png` (192x192 pixels)
- `icon-512.png` (512x512 pixels)

**Ferramentas gratuitas para criar:**
1. **Canva** (canva.com) - Crie um design 512x512
2. **Figma** (figma.com) - Crie um ícone profissional
3. **Favicon.io** (favicon.io) - Gera ícones automaticamente

**Dica de design:**
- Fundo azul (#0a4a8a)
- Ícone branco de cifrão ($) ou gráfico
- Texto simples e legível

## 🔐 SEGURANÇA E PRIVACIDADE

**IMPORTANTE:** Seus dados financeiros ficam apenas no navegador!

✅ **Dados seguros:**
- Nada é enviado para servidores externos
- Funciona 100% offline após instalado
- UPDATES fica no código do HTML

⚠️ **Para compartilhar com sua esposa:**

**OPÇÃO A: Mesmo arquivo (dados compartilhados)**
- Vocês dois acessam o mesmo link
- Qualquer update que um fizer, o outro vê
- Dados sincronizados

**OPÇÃO B: Arquivos separados (dados individuais)**
- Crie duas cópias do HTML
- Uma para você, outra para sua esposa
- Cada um tem seu próprio UPDATES
